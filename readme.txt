The project contains conecting to the database by key, downloading and preprocessing the data, creating dashboard using Tableau by technical requirements and creating a presentation.

Description of the files:

- Dashboard draft - the dashboard draft picture
- Dashboard link - link to the dashboard on Tableau public
- Database key - the Python key code for connection to the database
- Engine and preprocessing - Jupyter Notebook in .ipynb and .pdf formats with the project description, connecting to the database, preprocessing, creating "trending_by_time" and "requirements" files
- Presentation - the final presentation of the project
- Requirements - code requirements and libraries needed to downoad to get the code run properly
- trending_by_time - csv file containing the data after preprocessing (this file was used to create the dashboard)
